Artifacts

·      Sample User story:

-       As a car owner I want to schedule a service appointment.

-       As a car owner I want to cancel/ edit my scheduled service.

-       As a sales person I want to pre-qualify the buyer so we can determine the purchase limit.

-       As a user I want to be able to register online so that I can schedule/ book my car service.

-       As a dealership manager I want the customer to rate the service in order to improve future services.

·      Acceptance criteria:

-       User can only submit the form by filling all required fills.

-       The email the user provided must be validated.

-       User will receive a confirmation email/ text after successfully scheduling service.

-       An acknowledgment email is sent to the user after completing the form.

-       A user cannot submit a form without completing mandatory fill.

-       A user can only use free cleaning detail service if the car was purchased from dealership.

-       A user must be able to rate and add comment (maximum of 500 characters).
